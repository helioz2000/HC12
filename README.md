# HC12
various HC12 utilities & tools  

Australian **LIPD** permits the following:  
* 433.05 - 434.79MHz, max 25mW, all transmitters  
* 472.0125 - 472.1125, max 100mW, general telemetry  

Amateur Radio Band: 430-450MHz  

#### Power Settings
* 8 = 20dBm = 100mW  
* 7 = 17dBm = 50mW  
* 6 = 14dBm = 25mW  
* 5 = 11dBm = 12.5mW  
* 4 = 8dBm = 6.3mW  
* 3 = 5dBm = 3.15mW  
* 2 = 2dBm = 1.6mW  
* 1 = -1dBw = 0.8mW  

#### Channel Selection
Valid Channels: 1-127  
Min spacing between channels: 5

* Ch1 = 433.4Mhz (OK)  
* Ch2 = 433.8Mhz (OK)  
* Ch3 = 434.2Mhz (OK)  
* Ch4 = 434.8Mhz  
* Ch96 = 471.4MHz  
* Ch97 = 471.8MHz  
* Ch98 = 472.2MHz  
* Ch99 = 472.6MHz  
* Ch100 = 473.0MHz
